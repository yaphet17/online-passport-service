-----User guide line-----
1.First install XAMPP in your computer
2.Place OPS folder inside installedplace/XAMPP/htdoc
3.Open XAMPP control panel
4.Start Apache and MySQL server
5.Open PhpMyAdmin using "http://localhost/phpmyadmin/" in your browser adress field
6.Import ops.sql to phpmysql (ops.sql is provided in the ops/database folder)
7.Modify database configuration in config.php (if necessary)
8.Use "http://localhost/ops/index.php" or "http://localhost/ops/" to use as applicant
9.Use "http://localhost/ops/admin-login.php" to use as adminstrator
